Instructions

This assignment will be dedicated to test on using Apache Spark machine learning libraries. For this task, you can use the legacy RDD-based MLlib in pyspark.mllib and DataFrame-based MLlib pyspark.ml for Python. you need to work on Spark ML. You need to download your dataset, and analyze it using Spark ML libraries.

You will prepare a Colab notebook with analysis results.

Important note: The cells of the notebook should be executed and should have output. The cells which do not have an output or not executed or edited but not executed again will not be evaluated.

Each cell should have at least an execution order and an output (if gives an output).

You need to submit only the Colab notebook to Moodle.
